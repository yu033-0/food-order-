## 📖 程式介紹與規則
本系統採用 **C++ 物件導向 (OOP)** 技術開發，模擬餐廳自動點餐流程。
* **規則**：使用者可根據編號選購餐點，系統會即時計算小計與總額，並在結帳時產生完整的電子收據。
* **核心技術**：包含 5 個類別（MenuItem, Menu, OrderItem, Order, OrderSystem），實作了封裝與物件關聯。

## 🎮 玩法與操作說明
1. **顯示菜單**：程式啟動後自動顯示當日可供應餐點。
2. **選購餐點**：輸入數字編號選擇項目。
3. **設定數量**：輸入欲購買的數量。
4. **結帳清算**：輸入 `0` 結束點餐，系統會自動跳轉至「收據頁面」。
